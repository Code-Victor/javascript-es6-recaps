## Topics Covered

1. `Array.map`
2. `Array.filter` & `Array.find`
3. `Array.reduce`
4. Destructuring Arrays & Objects
5. Rest & Spread Operators (Arrays & Objects)
6. Dynamic Object Keys
7. `for...of` & `for...in` Loops
8. optional chaining
